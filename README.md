# AI Resume Parser System

## Overview

The AI Resume Parser System is a Python-based application that automatically extracts important information from resumes and converts unstructured resume data into a structured format.

The system helps simplify candidate screening by extracting key details such as name, email, phone number, skills, and educational qualifications from PDF resumes.

## Features

* PDF Resume Upload
* Automatic Text Extraction
* Name Detection
* Email Extraction
* Phone Number Extraction
* Skills Identification
* Education Detection
* CSV Report Download
* User-Friendly Interface

## Technologies Used

* Python
* Streamlit
* PDFPlumber
* Pandas
* Regular Expressions (Regex)
* Natural Language Processing Concepts

## Installation

Clone the repository:

git clone https://github.com/BK-BharathKumar7/AI-Resume-Parser.git

Navigate to the project folder:

cd AI-Resume-Parser

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
